FAQ

Question How do I withdraw my cryptocurrency?

You can withdraw your cryptocurrency by logging in, then going to the wallets page. You need to specify the address you want to send your cryptocurrency to and confirm that you are able to withdraw said currency.

Question How do I create my own Decentralized Creator?

You can review the source code on Github and then fork it and create your own. Please note that the software is licensed under Apache V2 and all future source code and development will hosted on eaxops.com in the near future.
